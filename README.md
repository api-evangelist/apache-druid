# Apache Druid (apache-druid)
Apache Druid is a high-performance, real-time analytics database designed for workflows where fast slice-and-dice analytics and instant data visibility matter. It supports both streaming and batch data ingestion with sub-second OLAP queries.

## APIs

### Apache Druid REST API
The Druid API provides REST endpoints for native JSON queries, SQL queries, data ingestion (both batch and streaming), datasource management, supervisor management, coordinator and overlord operations, and a JDBC driver for SQL access.
